# Tailor Book Pro

**Tailor Book Pro** is a web-based tailoring management system designed to help tailors and small tailoring businesses manage their daily operations digitally. The application allows users to store customer details, manage measurements, track orders, and monitor delivery status in an organized and efficient way.

## Project Overview

Tailor Book Pro replaces traditional paper-based record keeping with a modern digital solution. It helps tailors save time, reduce errors, and easily access customer and order information whenever needed.

## Features

* User authentication (Login / Registration)
* Add and manage customer details
* Store customer measurements
* Create and manage stitching orders
* Track order status and delivery dates
* User-friendly and responsive interface
